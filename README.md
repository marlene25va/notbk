# notebk - Personal Journal & Finance Tracker

A minimalist personal journal and financial tracking application built with React, TypeScript, and Vite. Available as both a web application and Android mobile app via Capacitor.

## Run Locally (Web)

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open http://localhost:3000 in your browser

## Build for Android

**Prerequisites:**
- Node.js
- Android Studio
- JDK 17 or higher

### Quick Start

Run the following command to build, sync, and open Android Studio:
```bash
npm run android
```

### Step-by-Step Process

1. **Build the web application:**
   ```bash
   npm run build
   ```

2. **Sync assets with Android project:**
   ```bash
   npm run android:sync
   ```

3. **Open in Android Studio:**
   ```bash
   npm run android:open
   ```

4. **Generate APK in Android Studio:**
   - Wait for Gradle sync to complete
   - Go to `Build` → `Build Bundle(s) / APK(s)` → `Build APK(s)`
   - Once complete, click on "locate" to find your APK file
   - The APK will be in `android/app/build/outputs/apk/debug/app-debug.apk`

5. **Install APK on your Android device:**
   - Transfer the APK to your device
   - Enable "Install from unknown sources" in your device settings
   - Open the APK file and install

### Generate Signed APK (for Release)

1. **Create a keystore:**
   ```bash
   keytool -genkey -v -keystore notebk-release.keystore -alias notebk -keyalg RSA -keysize 2048 -validity 10000
   ```

2. **Configure signing in Android Studio:**
   - Go to `Build` → `Generate Signed Bundle / APK`
   - Select `APK`
   - Choose your keystore file and enter credentials
   - Select `release` build variant
   - Click `Finish`

3. **Your signed APK will be in:**
   ```
   android/app/build/outputs/apk/release/app-release.apk
   ```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build web application
- `npm run preview` - Preview production build
- `npm run android` - Build, sync, and open in Android Studio
- `npm run android:build` - Build and sync with Android
- `npm run android:sync` - Sync web assets with Android project
- `npm run android:open` - Open Android project in Android Studio

## Project Structure

- `/` - React application source code
- `/dist` - Built web application (generated by Vite)
- `/android` - Native Android project (generated by Capacitor)
- `capacitor.config.ts` - Capacitor configuration

## Data Storage

All data is stored locally using localStorage. No backend or internet connection is required after installation.
